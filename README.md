# Safari Forever

Safari Forever was a community-driven auto platformer. Players conquered the campaign and enjoyed an infinite stream of community-created levels. They unleashed their own creativity and created levels that could be shared with friends effortlessly!

Created by Johannes Stensen. Original music and sound effects by [Gnooomen](https://soundcloud.com/Gnooomen).

This project is **archived**. The original author is no longer involved. See `LICENSE` for terms.

The community Discord still exists: [discord.gg/AVwnQgM](https://discord.gg/AVwnQgM)

## History

Development began in 2017. After 2.5 years of development, an alpha launched on **September 9, 2019** via Apple TestFlight and Google Play Alpha, followed by a busy period of community-driven iteration — ten patches in the first two months fixing physics edge cases, a full rewrite of the logic system (v0.73), cloud saving (v0.79), and the lock-and-key mechanic (v0.78).

The game launched publicly on the **App Store and Google Play on June 18, 2020**. Post-launch patches added an in-game level browser, push notifications, GIF recording, additional playable characters, new languages, and an endless mode. In September 2020 (v1.5) all in-app purchases were removed and the game became completely free.

The game was sunset in 2026. Two things happened at roughly the same time: Google removed it from the Play Store due to a security vulnerability in the version of Unity it was built on (Unity 2019.4) — fixing it would require stepping through seven major engine versions and rebuilding several dead third-party plugins from scratch — and the production server stopped booting for unknown reasons. Given the scope of work required versus the hosting cost and active player count, the decision was made to shut the game down and open-source the codebase.

The code is published here to preserve the history of the project, and in the hope that a technically inclined community member might one day use it as the foundation for a revival.

All 13,392 community-created levels have been preserved and are [available for download](https://drive.google.com/file/d/1UGUCbsiEplORwexyyKhf_Wu5FXNQGgsa/view).

  promo-material/  — App icon, promotional artwork, and YouTube links

## Building

This project does not compile in its current form. A number of commercial Unity plugins and proprietary assets were removed before open-sourcing and must be replaced or re-acquired separately. See `REMOVED_ASSETS.md` for the full list of what is missing, why it was removed, and where to find replacements.

No further support will be provided.
